(2024-09-26) — Fixing macOS Window Tiling Keyboard Shortcuts

The keyboard shortcuts in macOS for window tiling use the globe key, so they don't work with third party keyboards. The fix can be found in System Settings -> Keyboard -> Keyboard Shortcuts -> App Shortcuts. You can enter the names of each tiling function (which can be found in the "Window" section of the menu bar) and map them to you own shortcuts.
That's pretty complicated, so I figured out how to automate this from the terminal. The command below maps the keycodes cmd(@) + opt(~) + ctrl(^) + arrows(obscure unicode) for tiling to halves, adding shift($) for quarter tiling. I map caps lock key to all three modifiers using Hyperkey. You could also just choose simpler shortcuts if you don't want that. 

                        
defaults write -g "NSUserKeyEquivalents" '{
    Fill = "@~^f";
    Left = "@~^\U0090";
    Right = "@~^\U0092";
    Top = "@~^\U0091";
    Bottom = "@~^\U0093";
    "Bottom Left" = "@~^$\U2190";
    "Top Left" = "@~^$\U2191";
    "Top Right" = "@~^$\U2192";
    "Bottom Right" = "@~^$\U2193";
    "Return to Previous Size" = "@~^r";
}'
