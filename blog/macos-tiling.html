Fixing the Keyboard Shortcuts macOS Window Tiling

By Leaf Eriksen - September 25th, 2024

The shortcuts for the native window tiling in macOS use the globe key, so they don't work with third party keyboards.
The fix can be found in System Settings -> Keyboard -> Keyboard Shortcuts -> App Shortcuts -> +, where you can enter the names of each tiling function (which can be found in the "Window" section of the menu bar).
That's all a big pain the ass, so I spent over an hour figuring out a script to set this up from the terminal. This uses the key keycodes cmd(@) + opt(~) + ctrl(^) + arrows(obscure unicode). I map the caps lock all three modifiers using Hyperkey. 

                        
defaults write -g "NSUserKeyEquivalents" '{
    Fill = "@~^f";
    Left = "@~^\U0090";
    Right = "@~^\U0092";
    Top = "@~^\U0091";
    Bottom = "@~^\U0093";
    "Bottom Left" = "@~^$\U2190";
    "Top Left" = "@~^$\U2191";
    "Top Right" = "@~^$\U2192";
    "Bottom Right" = "@~^$\U2193";
    "Return to Previous Size" = "@~^r";
}'
